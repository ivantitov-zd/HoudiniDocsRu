= Reguide =

"""Разбрасывает новые направляющие, интерполируя свойства существующих направляющих."""

[Include:/shelf/reguide#includeme]

@parameters

Group:
        Разбрасывание направляющих осуществляется по указанной группе примитивов.

Density Scale:
        Множитель генерации точек на единицу площади.

Density Attribute:
        Атрибут, контролирующий плотность разбрасываемых точек.

Relax Iterations:
        Количество релаксаций позиций направляющих после разбрасывания. Более высокие значения приводят к более равномерному распределению направляющих за счет большего времени вычислений.

Only Display Roots:
        Не генерирует и интерполирует направляющие, а просто разбрасывает корневые точки. Поскольку генерация направляющих может быть довольно медленной, то можно использовать данный режим для быстрой визуализации и регулировки плотности.

Max Source Guides:
        Максимальное количество направляющих используемых для интерполяции каждой созданной направляющей.

Distance Bias:
        Управление силой влияния (весом) исходных направляющих на основе дистанции. Чем выше значение, тем быстрее влияние направляющих источников ослабевает с расстоянием. Значение 0 приводит к усреднению направляющих, без учета расстояния.
