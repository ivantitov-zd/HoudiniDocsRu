= Set Direction =

"""Направляет кривые по определенному вектору."""

Этот инструмент направляет кривые по определенному вектору.	Направление может быть задано с помощью векторного параметра или с помощью атрибута геометрии. Значение атрибута с ближайшей позиции к корню каждой кривой. Инструменты позволяют осуществлять отдельный контроль над вращением вокруг нормали поверхности и ее касательной (т.е. направление и подъем).

Нажав [Icon:FUR/comb] __Set Direction__ на вкладке __Guide Process__, будет создана нода [Guide Process SOP|Node:sop/guideprocess], и в параметре __Operation__ установлено значение __Set Direction__.  Для получения дополнительной информации см. справку [Guide Process SOP|Node:sop/guideprocess].
