= Reguide =

"""Интерполирует направляющие между существующими направляющими."""

Этот инструмент обычно используется после инструмента [Plant Guides|/shelf/plantguides], поскольку он интерполирует направляющие между существующими. Если создать несколько направляющих и причесать их, то можно использовать этот инструмент, чтобы быстро и легко заполнить пустые  места. Так можно быстро получить черновую шерсть на персонаже. Обычно используется для персонажей, которые полностью покрыты волосами.

== Using Reguide with Plant Guides == (includeme)

    # Нажмите на инструмент [Icon:FUR/plant_guides][Plant Guides|Node:sop/guidegroom] на вкладке __Grooming__.

    # Поместите несколько направляющих, нажав ((LMB)) на геометрии персонажа.
    
    # Причешите направляющие так, чтобы получить черновую форму шерсти, используя инструменты груминга.
    
        [Image:/images/fur/reguide1.png]   
    
    # Нажмите на инструмент [Icon:FUR/reguide][Reguide|Node:sop/reguide] на вкладке __Grooming__, чтобы заполнить поверхность направляющими, которые по форме и длине соответствуют существующим.
    
        [Image:/images/fur/reguide2.png]

    TIP:
        Скроллинг колеса мыши или удерживание ((Shift)) при перетаскивании ((LMB)) позволит вам изменить размер кисти.
        
    Для получения подробной информации о параметрах см. справку [Reguide SOP|Node:sop/reguide].
