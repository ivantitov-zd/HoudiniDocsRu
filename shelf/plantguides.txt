= Plant Guides =

"""Позволяет добавлять отдельные направляющие на поверхности персонажа."""

Этот инструмент позволяет добавлять отдельные направляющие на поверхности персонажа. Вы можете поместить несколько направляющих, причесать их, и волосы будут интерполированы между этими направляющими. Этот инструмент удобно использовать, если модель может измениться, поскольку это позволяет работать с небольшим количеством волос, которые можно легко пересадить. Его можно использовать для небольшого количества волос, таких как бакенбарды, или для персонажей, которые полностью покрыты волосами.

== Использование Plant Guides == (includeme)

    # Нажмите на инструмент [Icon:FUR/plant_guides][Plant Guides|Node:sop/guidegroom] на вкладке __Grooming__.

    # Поместите направляющие, нажав ((LMB)) на геометрии персонажа.
    
    [Image:/images/fur/plantguides.png]

    TIP:
        Скроллинг колеса мыши или удерживание ((Shift)) при перетаскивании ((LMB)) позволит вам изменить размер кисти.
        
    NOTE:
        Plant Guides интерполирует длину и форму направляющих, основываясь на окружающих направляющих. Если отсутствуют какие-либо направляющие, то первая направляющая будет перпендикулярна поверхности кожи сдлиной по умолчанию 0,1.

    Для получения подробной информации о параметрах см. справку [Guide Groom|Node:obj/guidegroom].
