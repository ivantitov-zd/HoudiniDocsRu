= Merge Groom Objects =

"""Объединяет направляющие и геометрии кожи нескольких объектов груминга."""

Часто бывает удобно создать груминг, используя отдельные объекты [Guide Groom|Node:obj/guidegroom], но неудобно [анимировать|Shelf:groom_animateguides] или [производить симуляцию|Shelf:groom_simguides] по отдельности. Этот инструмент объединяет отдельные объекты груминга из разных частей персонажа. После объединения можно добавлять ноды в один поток, и не беспокоиться об объектах груминга, подготовленных отдельно.

[Guide Deform|Node:obj/guidedeform], [Guide Simulate|Node:obj/guidesim], и [Hair Generate|Node:/obj/hairgen] могут выполняться после этой ноды, поэтому их необходимо использовать только один раз.

Это удобно при работе с разными частями персонажа. Зачастую в различных частях тела шерсть имеет разные направления, длину и плотность. Этот инструмент позволяет легко объединить отдельные части, создавая плавные переходы.

Для получения дополнительной информации см. справку [Работа с шерстью|/fur/workflow].

== Использование Merge Groom Objects == (includeme)

    # Нажмите на инструмент [Icon:FUR/groommerge]__Groom Merge__ на вкладке __Hair Utils__.

    # Выберите объекты груминга, которые следует объединить и нажмите ((Enter)), чтобы подствердить выбор.
    
    Для получения справки по определенному параметру см. справку по ноде [Groom Merge|Node:obj/groommerge].

NOTE:
    В качестве входных данных нода [Guide Groom|Node:/obj/guidegroom] также может иметь любую ноду груминга, включая [Groom Merge|Node:obj/groommerge].
    
TIP:
    Вы можете добавить несколько нод [Hair Generate|Node:/obj/hairgen] или [Guide Deform|Node:obj/guidedeform] после объединения, чтобы продолжить работу над отдельными частями.
