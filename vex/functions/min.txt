"""Возвращает наименьшее значение."""

:usage: `int min(int value1, int value2, ...)`
:usage: `float min(float value1, float value2, ...)`

    Возвращает наименьший из аргументов.

:usage: `<vector> min(<vector> value1, <vector> value2, ...)`

    Возвращает вектор, где каждый компонент является наименьшим из всех соответствующих компонентов аргументов.

:usage: `<type> min(<type> values[])`

    Возвращает наименьшее значение в массиве.

:usage: `float min(<vector> v)`

    Возвращает наименьший из компонентов вектора.

:usage: `<type> min(<type> value)`:
    Возвращает аргумент.
